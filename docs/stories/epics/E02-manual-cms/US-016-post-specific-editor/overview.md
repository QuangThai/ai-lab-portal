# Overview

## Current Behavior

Admin blog list links to a generic editor. Existing posts cannot be opened with their current data prefilled.

## Target Behavior

Admins can open `/admin/blog/[id]/edit` for a specific post, see its current title/slug/excerpt/content, and save/publish through existing server actions.

## Non-Goals

- No version history.
- No autosave.
- No AI workflow.
